Sémantique et formulaire

Présentation de la sémantique et des formulaires

Les dessous de HTML5

HTML ou HyperText Markup Language

En programmation, la sémantique fait référence au sens d'une partie de code. Sémantique signifie "qui se rapporte au sens". Écrire du code HTML sémantique signifie utiliser des éléments HTML pour structurer votre contenu en fonction de la signification de chaque élément, et non de son apparence.

Quand on pense à HTML, on pense souvent aux balises classiques : div, span, p. Elles fonctionnent, certes, mais elles sont souvent génériques.

Les formulaires HTML sont un des vecteurs principaux d'interaction entre un utilisateur et un site web ou une application. Ils permettent à l'utilisateur d'envoyer des données au site web. La plupart du temps, ces données sont envoyées à des serveurs web mais la page peut aussi les intercepter et les utiliser elle-même.

Pour concevoir une bonne page HTML, il faut suivre plusieurs étapes

  1. Programmer la page HTML
  2. Ajouter les assets sur la page
  3. Vérifier le code avec W3C

Il faut aussi prendre en compte :

  1. L'accesibilité
  2. Le RGPD
  3. L'éco-responsabilité
  1. Année de sortie officielle de HTML
    • Sous le nom de HTML 1.0
    • Créé par l'informaticien britannique Tim Berners-Lee
